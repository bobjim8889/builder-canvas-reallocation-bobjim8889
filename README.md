## Problem Statement

In AlgoArena's Agentic Workspace, a project canvas is a W \times H axis-aligned rectangle with corners at (0, 0) and (W, H).

One deployed widget already occupies a rectangular region on the canvas. Its sides are parallel to the canvas borders, with lower-left corner (x_1, y_1) and upper-right corner (x_2, y_2).

You want to place one more widget of size w \times h on the same canvas, with its width parallel to the canvas width.

Sometimes the new widget does not fit unless you shift the existing widget. Widgets are allowed to touch, but their interiors must not intersect.

You cannot rotate either widget, but you may move the existing widget anywhere inside the canvas.

For each test case, determine the minimum distance the existing widget must be moved so that there is enough room for the new widget, or report that it is impossible.

## Input

```
The first line contains a single integer t (1 \le t \le 5000) — the number of test cases.

For each test case, the first line contains two integers W and H (1 \le W, H \le 10^8) — the width and height of the canvas.

The second line contains four integers x_1, y_1, x_2, y_2 (0 \le x_1 < x_2 \le W; 0 \le y_1 < y_2 \le H) — the coordinates of the existing widget.

The third line contains two integers w and h (1 \le w \le W; 1 \le h \le H) — the width and height of the new widget.
```

## Output

```
For each test case, print the minimum distance you should move the existing widget, or -1 if there is no valid placement.

Your answer will be considered correct if its absolute or relative error does not exceed 10^{-6}.
```
